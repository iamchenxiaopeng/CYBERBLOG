# 2026-06-18 工作日志

## 修复两个 Bug（IP 访问时报错）

### Bug 1：`crypto.randomUUID is not a function`
- **原因**：`crypto.randomUUID()` 只在 HTTPS 或 localhost（Secure Context）下可用，通过 IP 地址访问时不可用
- **修复**：`guest.ts` 加 fallback — `typeof crypto.randomUUID === 'function'` 判断，不可用时改用 `Math.random()` 生成 16 位 hex 串
- **文件**：`frontend/src/utils/guest.ts`

### Bug 2：注册 500 Internal Server Error
- **原因**：`email` 字段有 unique 约束，多个用户不填邮箱时全部插入 null 导致唯一键冲突
- **修复**：`UserServiceImpl.register()` 中，邮箱为空时自动生成 `{username}@cyberblog.local` 占位邮箱；同时新增邮箱唯一性校验（仅真实邮箱时检查）
- **文件**：`backend/src/main/java/com/cyberblog/backend/service/impl/UserServiceImpl.java`

### Bug 3：POST /api/articles/upload 返回 403
- **原因**：JWT token 24h 过期后，前端仍带过期 token 发 POST 请求，Spring Security 返回 403，但前端没有感知"过期"
- **修复**（4 个文件）：
  - `stores/user.ts` — 新增 `isTokenExpired()` 解析 JWT payload 的 exp 字段判断过期，`isLoggedIn` 自动清理过期 token
  - `api/request.ts` — 写操作（POST/PUT/DELETE）遇到 403 时自动登出 + 跳转 `/login?expired=1&redirect=xxx`
  - `views/WriteView.vue` — onMounted 检查登录态，上传/发布前再次检查，过期则跳登录页
  - `views/LoginView.vue` — 登录成功后跳回 redirect 页面，显示"登录已过期"提示

### Bug 4：按钮 hover 后文字变黑看不清
- **原因**：`.btn-cyber:hover { color: var(--cyber-bg) }` 即 `#050508` 近乎纯黑，滑入动画背景未完全覆盖时黑字在深色背景上不可见
