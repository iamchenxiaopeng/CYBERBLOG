## 修复登录"网络错误" + 密码明文问题
- **根因**：数据库中存储的是明文密码，但后端用 BCrypt.matches() 验证 → 永远匹配失败 → 后端抛异常 → 前端显示笼统的"网络错误"
- **修复 1**：新建 `DataInitializer.java` — 首次启动自动创建 admin 账号（bcrypt(sha256) 双重加密）
- **修复 2**：新建 `init.sql` — 完整建表语句
- **修复 3**：改进前端 `request.ts` 错误处理 — 区分无响应/HTTP状态码/业务错误
- **修复 4**：新建 `fix-login.sh` 服务器诊断修复脚本

## 前端传输加密（SHA-256）
- **方案**：前端 SHA-256(password) → 后端 BCrypt(SHA-256) 存储
- **新增** `utils/crypto.ts`：Web Crypto API 优先，HTTP 环境降级 js-sha256 库
- **修改** `LoginView.vue` + `RegisterView.vue`：提交前 encryptPassword()
- **验证**：5 个测试用例（含中文）js-sha256 与 Node.js 标准完全一致 ✅
